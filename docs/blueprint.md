# **App Name**: DocuPilot

## Core Features:

- Image Resize: Resize images by specifying desired size in KB/MB and DPI.
- PDF Resize: Resize PDFs by specifying desired size in KB/MB and DPI.
- PDF Conversion: Convert PDFs to various formats: JPG, DOCX, Excel/CSV, PPT, PSD.
- Document Conversion to PDF: Convert various documents to PDF format: DOCX, Excel/CSV, PPT, PSD.
- PDF Manipulation: Merge PDFs, delete a page from a PDF, merge images and PDFs, edit PDFs directly.
- Passport Photo Maker: Create passport-sized photos from uploaded images.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F0F0F0) for a clean, modern interface.
- Accent color: Teal (#009688) for highlighting important actions and elements.
- Body and headline font: 'Inter', a sans-serif font, is recommended for its clean, modern look appropriate to technical documents.
- Use clear, intuitive icons to represent each document conversion type and feature.
- Maintain a clean and organized layout with a focus on usability and intuitive navigation.
- Use subtle transitions and animations to provide feedback on user interactions and guide the user through the document conversion process.
